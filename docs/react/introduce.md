```Silk```是基于React的前端组件库,服务于今目标移动前端业务。

## 安装
```bash
$ npm install jingoal-silk --save
```

## 使用
```js
import Button from 'jingoal-silk/lib/button';
ReactDOM.render(<Button>Start</Button>, mountNode);
```

## 体积

按需加载：只需引入业务中需要的组件即可，未 import 进来的组件不会打包进来。
